# Sistemas Tiziano ⚡ — Manual de Marca

---

## 1. Logotipo

### Versión Principal
**Sistemas Tiziano ⚡**
- El rayo (⚡) funciona como ícono de marca y reemplaza la "Z" en versiones animadas
- El rayo siempre va a la **derecha** del nombre completo en versiones estáticas

### Variantes de Uso
| Variante | Uso |
|----------|-----|
| `Sistemas Tiziano ⚡` | Logo principal para web, documentos, firmas |
| `ST ⚡` | Ícono/avatar para redes sociales, favicon |
| `Ti⚡iano` | Versión creativa para marketing y animaciones |

### Zona de Protección
Dejar siempre un margen mínimo de **1x** la altura del rayo alrededor del logo.

### Fondos Permitidos
- ✅ Fondo oscuro (`#0a0a0f`) con logo blanco + rayo amarillo — **uso principal**
- ✅ Fondo blanco con logo negro + rayo amarillo — versión clara
- ❌ Nunca poner el logo sobre fondos con textura o fotos sin overlay oscuro

## 2. Paleta de Colores

### Colores Primarios
| Nombre | Hex | Uso |
|--------|-----|-----|
| Negro Profundo | `#0a0a0f` | Fondos principales |
| Amarillo Eléctrico | `#FFD700` | Acento primario, rayo, CTAs, highlights |
| Naranja Eléctrico | `#FFA500` | Gradientes con amarillo |

### Colores Secundarios
| Nombre | Hex | Uso |
|--------|-----|-----|
| Azul Eléctrico | `#00B4D8` | Tags, links secundarios, acentos tech |
| Blanco Suave | `#F0F0F5` | Texto principal sobre fondo oscuro |
| Gris Medio | `#8892A0` | Texto secundario, descripciones |
| Gris Oscuro | `#555B68` | Texto muted, footnotes |

### Gradiente de Marca
```css
background: linear-gradient(135deg, #FFD700, #FFA500);
```
Usar en: botones primarios, subrayados, texto destacado.

### Reglas
- **Ratio mínimo de contraste**: 4.5:1 texto sobre fondo
- **El amarillo nunca como fondo completo** — solo como acento
- **Modo oscuro es la identidad principal** — el claro es complementario

## 3. Tipografía

### Fuentes
| Rol | Fuente | Peso | Fuente |
|-----|--------|------|--------|
| Títulos & Logo | **Outfit** | 700–900 | [Google Fonts](https://fonts.google.com/specimen/Outfit) |
| Cuerpo de Texto | **Inter** | 300–600 | [Google Fonts](https://fonts.google.com/specimen/Inter) |

### Escala Tipográfica
| Elemento | Tamaño | Peso |
|----------|--------|------|
| H1 (Hero) | 48–72px | 900 |
| H2 (Sección) | 32–48px | 800 |
| H3 (Card) | 20px | 700 |
| Body | 16–17px | 400 |
| Small/Tags | 12–14px | 500 |
| Labels | 13–14px | 600, UPPERCASE, letter-spacing: 3px |

### Reglas
- **Nunca usar más de 2 fuentes** en una misma pieza
- **Outfit solo para títulos y elementos de marca**, nunca para cuerpo
- **Inter para todo lo demás**: párrafos, botones, labels, descripciones

## 4. Iconografía & Elementos Gráficos

### Estilo de Íconos
- **Línea fina** (stroke-width: 1.5px)
- Tamaño estándar: 24px
- Color: `#FFD700` sobre fondos con opacidad dorada al 8%

### El Rayo ⚡
- Es el **elemento central de la marca**
- Siempre en amarillo `#FFD700` con glow sutil
- Animarlo cuando sea posible (pulse, flash, aparecer/desaparecer)
- Puede usarse solo como watermark o decoración

### Glassmorphism (Cards)
```css
background: rgba(255, 255, 255, 0.04);
border: 1px solid rgba(255, 255, 255, 0.08);
border-radius: 16px;
backdrop-filter: blur(8px);
```

## 5. Publicidad & Redes Sociales

### Tono de Comunicación
- **Profesional pero cercano** — imagen de startup, no de corporación
- **Confiable y técnico** sin ser inaccesible
- Tutear al cliente ("tu negocio", "contactanos")
- Evitar jerga técnica excesiva en comunicaciones públicas

### Posts para Redes
| Tipo | Ejemplo de formato |
|------|-------------------|
| Tip técnico | Fondo `#0a0a0f` + texto `#F0F0F5` + highlight `#FFD700` |
| Caso de éxito | Screenshot + overlay oscuro + logo en esquina |
| Promoción | Gradiente amarillo-naranja + texto negro + CTA blanco |

### Recomendaciones
- **Foto de perfil**: "ST⚡" sobre fondo `#0a0a0f`
- **Banner**: Nombre completo "Sistemas Tiziano ⚡" + tagline + gradiente sutil
- **Stories/Reels**: Siempre arrancar con el logo animado (Ti⚡iano → Sistemas Tiziano)
- **Consistencia**: Usar siempre la misma paleta oscura con acentos amarillos
- **Firma de email**:
  ```
  Sistemas Tiziano ⚡
  Desarrollo de Sistemas Web a Medida
  📍 Mar del Plata, Argentina
  📱 +54 11 3347-5296
  ✉️ adduci.tiziano@gmail.com
  ```

### Formatos Recomendados
| Pieza | Tamaño |
|-------|--------|
| Post Instagram/Facebook | 1080×1080px |
| Story | 1080×1920px |
| Banner LinkedIn | 1584×396px |
| Foto de perfil | 400×400px |
| Favicon | 32×32px y 180×180px (apple-touch) |

## 6. Aplicaciones del Logo

### Documentos & Propuestas
- Logo en header, alineado a la izquierda
- Fondo blanco, logo en negro + rayo amarillo
- Tipografía Inter para el cuerpo

### Tarjetas Personales
- **Frente**: Logo centrado sobre fondo `#0a0a0f`
- **Dorso**: Datos de contacto en `Inter` + rayo como detalle decorativo

### Presentaciones
- Slide de título: fondo oscuro, logo grande centrado
- Slides de contenido: logo pequeño en footer
- Títulos en Outfit, cuerpo en Inter
- Acentos en amarillo para highlights y datos clave

## 7. Slogan & Tagline

### Slogan Principal
**Energía digital, confianza real**

- Refuerza la identidad del rayo ⚡ como símbolo de energía e innovación.
- Comunica confianza y seguridad en el desarrollo de sistemas.
- Usar en: banner principal, firma de email, presentaciones.

### Reglas de Uso
- Tipografía: **Inter**, peso 400–500.
- Color: `#8892A0` (gris medio) sobre fondo claro, `#F0F0F5` (blanco suave) sobre fondo oscuro.
- Ubicación: siempre debajo del logo o en el área de tagline definida.
